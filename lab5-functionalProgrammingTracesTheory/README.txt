Zadanie zostało napisane w języku Scala. Program można uruchomić metodą sbt run <nazwa_pliku wejściowego>

Plik wejściowy musi być umieszczony w folderze src/main/resources

example_output*.txt to wynik dla odpowiednich plików example_input*.txt, które są też umieszczone w src/main/resources, a więc
można testowo uruchomić program dla tych plików wejściowych -> sbt run example_input1.txt
